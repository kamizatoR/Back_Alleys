# Pets

## サイト概要
### サイトテーマ
ペットや動物を画像やテキストで共有するSNS

### テーマを選んだ理由
ペット（動物）に癒されたい人や飼っているペットを他の人にシェアしたい、思い出として残しておきたい人などが利用できるサービスがあったらよいと思い着手いたしました。

### ターゲットユーザ
動物が好きな人・動物を見て癒されたい人

### 主な利用シーン
動物を見て癒されたい時やペットを共有したい時など
